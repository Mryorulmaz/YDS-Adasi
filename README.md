# YDS Adası 🏝️
A comprehensive vocabulary learning and exam preparation platform developed with **Flutter**.

## 📌 Overview
YDS Adası is designed to help students prepare for academic language exams with a focused database and interactive learning tools. The application manages complex data structures and provides a seamless user experience for language acquisition.

## ✨ Key Features
- **Large Academic Database:** Curated list of 5000+ words specifically for exam preparation.
- **Dynamic Quiz System:** Interactive testing modules and progress tracking.
- **In-App Subscriptions:** Integrated Apple In-App Purchases (IAP) for premium feature access.
- **Monetization:** Scalable Google AdMob integration for advertisement-based revenue.
- **Optimized UI/UX:** Clean and focused design to enhance learning efficiency.

## 🛠 Tech Stack
- **Framework:** Flutter (Dart)
- **State Management:** Provider / Riverpod (Hangisini kullandıysan onu bırak)
- **Monetization:** Google AdMob & Apple In-App Purchases
- **Backend Services:** Firebase Integration
- **Platform:** iOS (App Store)

## 🚀 Development Highlights
- Implemented **In-App Purchase (IAP)** flow for digital content delivery.
- Managed **monetization strategies** through ad-mediation and subscriptions.
- Designed and maintained a production-ready application on the **App Store**.
- Optimized local data handling for large word lists.
